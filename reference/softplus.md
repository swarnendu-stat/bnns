# softplus transformation

softplus transformation

## Usage

``` r
softplus(x)
```

## Arguments

- x:

  A numeric vector or matrix on which softplus transformation is going
  to be applied.

## Value

A numeric vector or matrix after softplus transformation.

## Examples

``` r
softplus(matrix(1:4, nrow = 2))
#>          [,1]     [,2]
#> [1,] 1.313262 3.048587
#> [2,] 2.126928 4.018150
```
